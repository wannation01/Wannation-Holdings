# Wannation Holdings Brand Assets

This directory contains all official brand assets for Wannation Holdings.

## Contents

- **logos/** - Official logo files in various formats
- **colors/** - Color palette specifications
- **guidelines/** - Branding and usage guidelines

## Logo Files

### Available Formats
- SVG (Scalable Vector Graphics) - Recommended for web and print
- PNG (various sizes) - For web and digital use
- PDF - For print materials

### Logo Descriptions

1. **Bull Logo** (`bull-logo.*`)
   - Primary brand identity
   - Red color scheme
   - Represents strength and innovation
   - Contains musical note symbol

2. **Skull Logo** (`skull-logo.*`)
   - Secondary branding
   - Gray/military color scheme
   - Represents power and protection
   - Wings for forward momentum
