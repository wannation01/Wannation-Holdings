# Wannation Holdings Brand Guidelines

## Overview

The Wannation Holdings brand identity consists of two primary logos, each representing distinct aspects of our organization's values and mission.

---

## Logo System

### 1. Red Bull Logo
**Primary Use:** Main brand identity, marketing materials, primary communications

**Description:**
- Bold red bull head with a musical note in a triangular warning symbol
- Represents strength, energy, and creative innovation
- Color: Vibrant red (#FF0000 or similar)
- Background: Works on both light and dark backgrounds

**Dimensions:** 
- Recommended minimum size: 100px
- SVG format for scalability

**Usage:**
- Company header/branding
- Marketing materials
- Social media profiles
- Primary touchpoints

---

### 2. Gray Skull with Wings Logo
**Primary Use:** Secondary branding, special projects, protection-focused initiatives

**Description:**
- Stylized gray skull with wings in a shield
- Represents power, protection, and forward movement
- Color: Military gray/dark olive
- Background: Works excellently on dark backgrounds

**Dimensions:**
- Recommended minimum size: 100px
- SVG format for scalability

**Usage:**
- Secondary branding elements
- Special project identification
- Security/protection-related communications
- Badge systems

## File Formats

All logos are available in:
- **SVG** - Scalable Vector Graphics (recommended)
- **PNG** - For web (various sizes)
- **PDF** - For print materials

---

## Do's and Don'ts

### ✅ Do's
- Use official logo files provided
- Maintain clear space around logos
- Use appropriate colors as specified
- Scale proportionally
- Use high-quality files for print

### ❌ Don'ts
- Do not alter colors without approval
- Do not distort or stretch logos
- Do not add effects without guidelines
- Do not use low-resolution versions for print
- Do not combine logos in unauthorized ways
